Создаем 1 файл:
cat > task1.txt << EOF
line1
line2
line3
line4

EOF
Создаем 2 файл:
cat > task1.2.txt << EOF
line6
line7
line8
line9
line10
EOF
Создаем объединяющий файл:
cat task1.txt task1.2.txt > task_total.txt

cat task_total.txt

- скопируйте файл, дав ему новое имя.

Задание2.  В ОС Linux скрытыми файлами считаются те, имена которых начинаются с символа точки .. 
Напишите команду, которая будет показывать количество скрытых файлов в домашнем каталоге пользователя. 
При необходимости используйте конвейер (пайплайн). 
Подсказка: для подсчета количества строк можно использовать wc). 
Количество скрытых файлов во вложенных папках считать не нужно.
find -maxdepth 1 (только из текущей директории) -type f(файлы) -name ".*"  | wc -l (кол-во строк)
